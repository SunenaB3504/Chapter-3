🪄 Magic Rules (Properties of Multiplication) 🪄

Multiplication has some cool rules that make it easier!

Rule 1: Order Doesn't Matter (Commutative Property)

Multiplying numbers in any order gives the same answer.

5 x 64 is the same as 64 x 5.

2 x 3 = 6 and 3 x 2 = 6

Rule 2: Grouping Doesn't Matter (Associative Property)

When multiplying three or more numbers, you can group them differently, and the answer stays the same.

(6 x 5) x 130 is the same as 6 x (5 x 130).

(2 x 3) x 4 = 6 x 4 = 24
